Practical Assignment

The goal of this practical assignment is to implement a small rich-text renderer which takes JSON input and produces HTML output. The JSON-based input format has been customly designed for this assignment and it is one of your tasks to infer its schema and comprehend its semantics.

Your Tasks

  1. Take a brief look at the JSON in the "input" section of this playground and compare it to the expected output (this text). Feel free to toggle between the expected HTML and its visual preview.
  2. A short, informal specification of the input format follows. Make sure to read it thoroughly.
  3. Design a sensible TypeScript typing for the input format. Write all of the type definitions into "/src/render.ts". Try to cover all of the specified restrictions; make use of the newly-gained knowledge of advanced typing features.
  4. Implement the body of the "render()" function so that it produces (more-or-less) the same output as you are reading right now.

Input Format

The input JSON represents the content of an article as an array of blocks. There are 3 types of blocks: headings, paragraphs and lists. Each block carries its type (for easy differentiation) along with other properties.

Headings

A heading is characterized by:

  • its text content – an arbitrary string;
  • a level – an integer between 1 and 6 (inclusive).

Paragraphs

A paragraph only has a text content (again, an arbitrary string).

Lists

Each list carries:

  • an array of items (arbitrary strings);
  • a flag denoting whether it is ordered (numbered) or unordered (bulleted).

In addition to that, every ordered list (and no unordered one) has 1 of 2 numberings:

  1. numeric;
  2. alphabetic.
